# RelayTV Asset and Trademark Policy

This file describes how RelayTV project assets are licensed and how the
RelayTV name and visual identity may be used.

## Code and Documentation

## RelayTV Marks

The RelayTV name, logo, banner, splash images, badges, and other brand identity
assets are project marks.

You may use RelayTV marks to:

- identify the RelayTV project truthfully
- link to the RelayTV project
- describe compatibility with RelayTV
- show unmodified screenshots of RelayTV in reviews, documentation, tutorials,
  or issue reports

You may not use RelayTV marks to:

- imply endorsement by RelayTV or its maintainers
- present a modified project as the official RelayTV project
- use RelayTV marks as the primary branding for a different product or service
- remove or alter notices attached to bundled third-party assets

## Bundled Project Artwork

Project artwork in these paths is distributed with RelayTV for use by the
RelayTV application and documentation:

- `app/relaytv_app/static/brand/`
- `docs/images/`

These assets may be copied and displayed as part of running, documenting,
reviewing, or redistributing RelayTV. Trademark-style use of the RelayTV name
or marks remains subject to the mark rules above.

## Weather and UI Icons

Weather and UI assets bundled under `app/relaytv_app/static/` are included so
RelayTV can render its idle dashboard and web UI without external asset
dependencies at runtime.

If a bundled asset originates from a third-party source, its upstream license
and attribution should be documented in `THIRD_PARTY_LICENSES.md` or the
release-time third-party inventory.

## Screenshots

Screenshots in `docs/images/` are documentation assets. They may be reused to
describe RelayTV, demonstrate RelayTV behavior, or discuss RelayTV features.
Do not use screenshots in a way that implies endorsement of another project,
service, or commercial offering.

## Questions

If an asset's license or permitted use is unclear, treat it as reserved until
the license is clarified in this file or in `THIRD_PARTY_LICENSES.md`.
